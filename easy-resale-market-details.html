About The Project

  1. Its online product resale website
  2. Here are thrree type of users. They are admin, buyer, seller
  3. Admin can make a user as admin and can manage other users
  4. Sellers can add products, manage them, and can advertise them. Admin can verify any specific seller.
  5. By this website a user can sell/buy his/her product.
  6. By the dashboard of this website admin can manage its ueser.
  7. Users can book and pay for products. Here Stripe gateway payment system is integrated.
  8. User authentication and user authorization is implemented using firebase auth and JWT
  9. Technologies: React.js, Express.js, MongoDB, REST API, Tailwind CSS, Firebase Auth, Stripe, JWT
